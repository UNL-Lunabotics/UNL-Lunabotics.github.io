# UNL Lunabotics Documentation

Welcome to the official documentation hub for the **University of Nebraska–Lincoln Lunabotics Team**.  
This site provides technical references, design notes, development standards, and project information for current and future team members.

---

## Overview

The UNL Lunabotics Team participates in NASA’s Lunabotics Challenge by designing, building, and testing an autonomous lunar excavation rover.  
Our goals include:

- Designing reliable mechanical systems for excavation and mobility  
- Developing robust embedded software for navigation and control  
- Implementing autonomous operations for on-field excavation tasks  
- Conducting rigorous verification and testing throughout the build cycle  

This documentation serves as a central reference for students contributing to the project.
