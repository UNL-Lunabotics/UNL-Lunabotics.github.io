## Intel Realsense Cameras

The setup for most of the Realsense camera family should be relatively the same. At the time of writing this, UNL Lunabotics uses the Intel Realsense D345i's. These cameras are depth-sensing and have a built in IMU.

## Installation

1. Ensure the required software library for ROS2 is installed: `sudo apt install ros-jazzy-realsense2-camera`.
